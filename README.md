### Installation
1 - Clone the repository
```bash
    git clone https://github.com/bil-elmoussaoui/TwoFactorAuth && cd ./TwoFactorAuth
```
2 - Install `PyOTP`
```bash
    sudo pip install pyotp
```
3 - Afterwards
```bash
./autogen.sh
make
sudo make install
```
4 - You can run the application from your terminal using 
```bash
twofactorauth
```
